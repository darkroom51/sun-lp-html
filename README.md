# sun-lp

Some cool landing page layout.

### Installation

```
npm install
```

### Start Dev Server

```
npm start
```
App will be loaded on `http://localhost:8080/`

### Build Prod Version

```
npm run build
```
Prod files will be stored in /dist folder
